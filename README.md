# 처음 해보는 Servlet & JSP 웹 프로그래밍

📘 처음 해보는 Servlet & JSP 웹 프로그래밍 (오정임 지음) 실습 예제 저장소입니다.

## 📁 프로젝트 개요
마지막 챕터인 16장의 회원 관리 CRUD 프로젝트 파일입니다.  
Spring Boot 환경에서 내장 Tomcat을 사용해 실행하며, Gradle 기반으로 구성되어 있습니다.

## 🛠️ 개발 환경
- Java 17  
- Spring Boot (내장 Tomcat)  
- Gradle  
- IntelliJ IDEA  
- JSP / Servlet / JSTL / EL
